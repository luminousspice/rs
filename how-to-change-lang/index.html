Anki2の言語設定を変更する

2013-04-28

Anki2では、初回起動直後の言語設定以後、言語設定のユーザーインターフェイスを用意していません。起動オプションで言語を指定する方法と、初期設定ファイルを削除して言語設定をやり直す方法の手順を紹介します。

はじめに

Anki2からインストール直後の初期設定でユーザーインターフェースの使用言語を選択した後、設定画面から言語設定を変更することができなくなりました。複数の言語を学習にAnkiの言語設定を切り替えながら使ってきたユーザーの方々は、この変更点にとまどっているのではないでしょうか。

そこで、今回はインストール後にAnkiの言語設定を切り替える方法をまとめて紹介します。

一時的に変更

Anki2からはStartup Options(起動オプション)で言語設定を指定できるようになりました。

インターフェースの言語は"-l"オプションで指定できます。-l <ISO言語名コード>を指定して起動します。

例: Windowsで英語を指定して起動

"C:\Program Files\Anki\anki.exe" -l en

例: Mac OS Xで日本語を指定して起動

/Applications/Anki.app/Contents/MacOS/anki -l ja

実際には皆さんお使いのAnkiのインストール場所を指定してください。

NOTE
使用可能な言語

Ankiマニュアルでは、"iso country code" と記述していますが、正しくは言語名コード(Language code)です。ISO 639に掲載している全ての言語で使用できるとはとても思えないので調べてみました。

使用可能な言語は、Ankiのソースコードanki/lang.pyの先頭で定義しています。言語リソースの翻訳状況は様々ですがAnki2.0.8時点で、50言語 サポートしています。これ以外の言語を指定した場合は英語で代用します。

コマンドプロンプトから毎回起動オプションを使うのは面倒な場合もあります。でどうするかというのがこの先の話題です。

Windows のショートカットを利用

  1. anki.exe のショートカットを右クリックし、プロパティを選択
  2. ショートカットタブを選択
  3. リンク先項目の最後に起動オプションを追加する。例: "C:\Program Files\Anki\anki.exe" -l en

初期設定し直す

インストール直後の初期状態に戻して言語設定し直すというのが次のアプローチです。Ankiの初期設定ファイルを削除すれば良いのです。Ankiの再インストールでは、初期設定はそのまま残りますので、言語設定し直すことができません。

Ankiは既定でドキュメントフォルダーの中に単語帳を保存するAnkiフォルダーを作成します。この中にある prefs.db (Documents/Anki/prefs.db) が初期設定ファイルです。Ankiを終了してから prefs.db を削除し、Ankiを起動すると言語設定し直すことができます。 当然のことながら他の設定もなくなってしまいます。

より安全に作業するには、Ankiを終了してから prefs.db を別のファイル名に変更します。元のファイルの名前を prefs.db に変更すればいつでも元の設定に戻すことができます。

頻繁ではないにしても継続的に言語設定を切り替えるのであれば、あらかじめ言語別に prefs.db を保存しておいて使い分けるのも効率的な方法だと思います。

Mac OS Xの場合の保存場所

[書類]>[Anki] prefs.db (/Users/[ユーザー名]/Documents/Anki/prefs.db)

Windows7の場合の保存場所

[マイ ドキュメント]>[Anki] prefs.db ("C:\Users\user name\Documents\Anki\prefs.db")

Windowsの場合バージョン毎にドキュメントフォルダーの位置が異なります。Windows XP または Windows Vista からのアップグレード後のファイルおよびフォルダーの場所の「Windows 7 で共通のフォルダーを見つける方法」を参考にしてください。

注意: ユーザープロフィールを複数使っている場合

この方法を行うと、既定で作成するプロファイル"ユーザー 1"以外のプロファイルが消えてしまいます。ここで慌てないでください。再度、これまで使っていたプロファイル名でプロファイルを作成すると元に戻ります。プロファイル名を忘れてしまった方は、ドキュメントフォルダー内のAnkiフォルダーを見てください。プロファイル名がフォルダー名として残っています。

設定ファイルを編集

prefs.db はsqliteファイルです。開発経験が必要になりますが、sqliteファイルを書き換えることでも言語設定は変更できます。

どう使い分けるか

まず最も頻繁に使う言語にAnkiの言語設定をすることです。現在そうなっていないのなら、初期設定からし直すのがよいでしょう。その上で一時的に使う言語は起動プションで呼び出すのがおすすめの方法の方法です。
