Anki 単語帳サンプル一覧

2014-04-15

これまでの投稿で公開してきた Anki 単語帳のサンプルを紹介します。既定のテンプレートを拡張した機能を含んでいます。新しい教材をつくる時の参考として役立て下さい。

辞書を開くリンクと読み上げ機能

Ankiのカード表示を編集するでは、カードにフィールドの文字を辞書検索するリンクを追加する方法、アドオン AwesomeTTS を使って読み上げる方法を説明しました。

図 1. カード表示例

Mac OS X の場合は、内蔵の辞書アプリからテキストを検索して表示します。

Anki 単語帳パッケージ how2anki4.apkg

解答キー入力機能と空欄補充

Anki解答キー入力機能の使い方では、解答をキー入力するカードのつくり方、さらに空欄補充問題と組み合わせた応用例を紹介しています。

図 2. カードのプレビュー

空欄補充問題の解答をカードの穴埋め欄に記入するカードです。解答を表示すると入力内容と正解の比較ができます。

Anki 単語帳パッケージ anki209.apkg

Widows 開発者のための頻出語彙 MSDN Word List

Windows開発者の必修語彙をAnki学習する方法では Windows アプリケーション開発者が技術文書を読むための語彙集を作成しました。 一般的な英語学習者が取得すべき基本語彙2570語(GSL+AWL)に追加して覚えれば、MSDN Magazineの使用語彙の98%がカバーできる語彙824を収録しています。 なお、この分析には固有名詞は除外して計算しています。

表 1. 単語帳の内訳
子単語帳名 単語数 内容
MSDNWords::Level1 179 基本語彙(GSL+AWL)との組み合わせで使用語彙の95%までカバーできる単語(カタカナ、略語を除く)
MSDNWords::Level2 276 基本語彙(GSL+AWL)との組み合わせで使用語彙の98%までカバーできる単語(カタカナ、略語を除く)
MSDNWords::Level3 102 過去10年分のバックナンバーまで分析対象を拡大した時、新たに追加になった頻出語彙
MSDNWords::カタカナ英語 223 カタカナ英語として使われている単語
MSDNWords::略語 50 訳語をあてずに使われている略語
図 3. 単語帳使用例

解答面には、MSDNオンライン検索(MSDN MagazineとMSDN Library限定)、マイクロソフト用語集へのリンクを設定していますので、実際に使用している文例を確認することができます。

Anki 単語帳パッケージ msdnwl.apkg

WordPress 開発者のための WordPress Codex 頻出語彙

WordPress Codexの英語 WordPress開発者のためのAnki単語帳では、WordPress 開発者が技術文書を読むための語彙集を作成しました。 一般的な英語学習者が取得すべき基本語彙2570語(GSL+AWL)に追加して覚えれば、WordPress Codex の使用語彙の98%がカバーできる語彙445を収録しています。 なお、この分析には固有名詞は除外して計算しています。

表 2. 単語帳の内訳
タグ名 単語数 内容

WPCX2 241 頻出2000語と学術語彙(AWL)との組み合わせで使用語彙の98%までカバーできる単語
図 4. 単語帳使用例

解答面には、WordPress Codexへのリンクを設定していますので、実際に使用している文例を確認することができます。
